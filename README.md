# Goddess
一个看妹纸与开发资讯的Android APP，新手向大神学习的练手项目，来自代码家的API http://gank.io。

模仿系统相册练习的时候顺便搞出来的相册. 数据来自 [http://gank.io/](http://gank.io/) by [@daimajia](https://github.com/daimajia).

API of gank.io: [http://gank.io/api](http://gank.io/api).
